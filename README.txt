Deploy Static Website on AWS

In this project, I deployed a static website to AWS using S3, CloudFront, and IAM.

Entry Url: http://aws-static-blog.s3-website-us-east-1.amazonaws.com/
Cloudfront URL: http://d3hqznnl28byhs.cloudfront.net/

The files included are: 

index.html - The Index document for the website.
404.html - Error page
atom.xml - RSS feed
favicon - icon for the site
search.xml - supplement for built in search functionality
/images - The background image file for the website.
/js - JavaScript libraries needed for the website to function.
/main - JavaScript/css libraries needed for the website to function.
/css - CSS files for the website.
/2020 - Blog files, archived by year
/about - about page
/archive - archive page
/categories - categories page
/projects - project page
/tags - tag page
/screenshots - screenshot the results of each step
